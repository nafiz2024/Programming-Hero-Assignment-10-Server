# PromptFlow Server

Minimal Express backend scaffold for the PromptFlow Server project.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create your environment file from the example:

```bash
copy .env.example .env
```

3. Update the port in `.env` if needed.

## Run

Start in development mode:

```bash
npm run dev
```

Start in production mode:

```bash
npm start
```
